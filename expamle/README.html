修士論文クラス ampmt.cls の説明書 （2019/04/01）

はじめに

ampmt.cls は数理工学専攻の修士論文を書くためのクラスファイルです． ampmt.cls を用いると，様式が指定されている表紙，背表紙，摘要，目次が必要項目を入力するだけで適切に生成されます．また，サンプルファイルで論文の形式の雛形も与えています．

準備

ダウンロードしてきた ZIP ファイルを作業用のディレクトリ（修士論文の原稿ファイルを置くフォルダ）で解凍します．以下のファイルが生成されます．

  • README.html：このファイル．
  • ampmt.cls：クラスファイル本体．
  • ampmt_sample_japanese.tex, ampmt_sample_japanese.pdf：和文の場合のサンプルソースとその出力．
  • ampmt_sample_english.tex, ampmt_sample_english.pdf：英文の場合のサンプルソースとその出力
  • ampmt_figure.pdf, ampmt_figure.tex：サンプルファイル中に挿入されている図とそのソース．
  • logo.pdf：表紙に挿入される京都大学エンブレム．

文字コードは全て UTF-8 に統一されています．エディタ等で文字化けが生じた場合は UTF-8 に対応しているかどうか確認して下さい．

使い方

和文の場合は ampmt_sample_japanese.tex を，英文の場合は ampmt_sample_english.tex を同じディレクトリ内にコピーし，適当な名前に変更します．エディタで開いてプリアンブルに題目等の項目をコメントの説明に従って記入します．その後，本文の執筆にとりかかって下さい．

タイプセットは通常通り platex + dvipdfmx （和文）や pdflatex（英文） を用いればよいです．万一 platex が UTF-8 のコードをうまく認識しない場合は -kanji=utf8 を platex のオプションに指定して下さい：

    platex -kanji=utf8 thesis.tex

英文で platex + dvipdfmx を使いたい場合は dvipdfmx グローバルオプションをつけるのを忘れずに．

出力は，中表紙，摘要，目次，本文，表紙，背表紙，提出用摘要の順となります．印刷のうえ，中表紙から本文までは2穴パンチで穴を空けて，配布されたバインダーに綴じて下さい．表紙，背表紙は適切に加工のうえバインダーに糊で貼り付けて下さい．できた論文のバインダーに提出用摘要を添えて，情報学研究科事務室の教務掛に締切までに提出して下さい．提出する前に題目や各人の氏名，年月日等が合っているかどうか必ず確認すること．

注意点として，本文を両面印刷とする場合でも，本文以外については片面印刷にする必要があります．クラスオプションに twoside を指定すると，これらのページの後ろに空白ページが適切に挿入され，両面印刷設定でそのまま印刷できるようになります．

よくあるかもしれない質問と回答

一般的なエディタや LaTeX の使い方については各自で書籍や Web ページ，優秀な友人や後輩にあたって下さい．処理中にエラーが出た場合は，まずはエラーメッセージを読んで何が起きているのか理解することを試みましょう（英語だからと言って思考停止しない）．

Q：末尾の表紙，背表紙，提出用摘要が出力されないように（もしくはこれらだけが出力されるように）したいです．
A：サンプルの \begin{document} の直前にある「出力の制御」のところをいじって下さい．
Q：フォントサイズを変えたいです．
A：デフォルトでは 11pt になっていますが，通常通りクラスオプションに 12pt と指定すれば大きくできます．10pt と小さくすることもできますが，小さな文字は読みにくいので推奨しません．
Q：\chapter コマンドを使うと ! Undefined control sequence. と出ます．
A：デフォルトでは jsarticle や article をベースに用いるため，\chapter は未定義でエラーが出ます．\chapter を使いたい場合はクラスオプションに chapter を指定すると jsbook[report] や report をベースに用いるようになります．
Q：英文で ! LaTeX Error: Cannot determine size of graphic in logo.pdf (no BoundingBox). と出ます．
A：英文のサンプルはデフォルトでは pdflatex を使うようになっており，platex などでタイプセットしようとするとこのエラーが出ます（platex では PDF を直接扱えないため）．クラスオプションに dvipdfmx を指定すると正しく動作するようになります．
Q：\vspace{...} などで指定した寸法よりも若干大きなスペースが入っているようですが．
A：これは jsclasses をベースに用いているために起きる現象です．フォントサイズが 11pt の場合，指定した寸法を1.1倍した値になってしまいます．これを回避するためには，単位 xx の代わりに truexx を用いて下さい（2truemm など）．
Q：レイアウトを変えるために geometry パッケージを読み込むと ! LaTeX Error: Option clash for package geometry. と出ます．
A： ampmt.cls 内で既に geometry を読み込んでいるために，プリアンブルにオプション付きで geometry を読み込むと option clash となります．\begin{document} の直前に書いてあるように，\geometry コマンドを使って下さい．
Q：英文で ! LaTeX Error: File `newtxtext.sty' not found. と出ます．
A：どうやらお使いの TeX システムが少し古いため，newtx が入っていないようです．これはただのフォント設定のサンプルなので，該当行を消して Computer Modern を使うなり，お好みのフォントのパッケージに書き換えるなり，TeX システムをアップデートするなりして下さい．
Q：platex に -kanji=utf8 オプションをつけましたが platex: unrecognized option '-kanji=utf8' と出ます．
Q：\numexpr という書いた覚えのない命令に対して ! Undefined control sequence. と出ます．
A：どうやら pTeX のバージョンが古すぎるようです．最新の TeX システム（TeX Live など）を導入しなおして下さい．
Q：uplatex を使いたいのですが，! LaTeX Error: Font family `JY1+mc' unknown. と出ます．
A：クラスオプションに uplatex を指定して下さい．
Q：xelatex や lualatex を使いたいのですが，! LaTeX Error: This file needs format `pLaTeX2e' but this is `LaTeX2e'. と出ます．
A：jsclasses をベースに使っているので，platex, uplatex 以外ではエラーが出ます．今のところ xelatex や lualatex に対応する予定はありませんが，どうしても使いたい場合は自身で ampmt.cls に手を入れて下さい．
